Admin Area

This is the area where you can add or change existing items, ship out orders or make packing slips


To add an item or update it with new information, select: "ADD ITEM" from the admin menu.


To ship an order, select: "ORDERS+" from the admin menu and select "SHIP" in actions on the Order you would like to ship.


To print a packing slip for an order, select: "ORDERS+" from the admin menu and select "Pack-list" in actions on the Order you would like to ship.
